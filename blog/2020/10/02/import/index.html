Data Importation in R

By Indra Bishnoi | December 9, 2020

In this tutorial, I will teach you how to import your data into R from a multitude of sources. Importing your data correctly can help you prevent several time consuming mistakes in the future. Before you begin, remember to put your file into your working directory.

For importation, you will need to install the following packages, respectively.

install.packages("tidyverse")
install.packages("readxl")
install.packages("foreign")

After you’ve downloaded the necessary package, you’re ready to import your data!

From a comma-separated values file

Say you’d like to read in a file named “data” and it is a csv file so the full name is “data.csv”. Load the tidyverse package like so:

library(tidyverse)

Then, read in your .csv file using <read_csv>, and name it whatever you’d like (I chose df)

df <-read_csv("./data.csv")

From excel

After you’ve put your csv file into your working directory (e.g. data.xlsx), one method to import this into R’s global environment is by using the xlsx package. Start by loading in this package like so:

library("readxl")

Then, read in your file using <read_excel>. I have opted to save mine as a dataframe called df:

df <-read_excel("./data.xlsx")

From SPSS

A number of labs that previously used SPSS are now transitioning over to R. When importing a SPSS dataframe to R (e.g. data.sav), put it into your working directory and then use the foreign package:

library(foreign)
df <- read.spss("data.sav", use.value.label=TRUE, to.data.frame=TRUE)

Here, I saved my SPSS file into my global environment with the name df and used the <read.spss> command to import. <use.value.label=TRUE> converts variables with value labels in SPSS into R factor. <to.data.frame=TRUE> makes the df a data frame. If it was not defined/FALSE (default), the df would be a list instead of a data frame.

From a package

Someimtes, packages come with their own dataframe. For example, tidyverse has a fun starwars data frame which can be used as a sample data set (named df). To load this:

library(tidyverse)
df <- starwars
