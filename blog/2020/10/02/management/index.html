Data Management in R

By Indra Bishnoi | December 9, 2020

In this tutorial, I will teach you how to manage your data in R. Specifically, how to view and select your data, and how to deal with missing data.

Before you begin, remember to import your data.

Viewing your data

If your data frame is called df, then use the <str()> function to view it’s structure.

str(df)

To view the first six lines of df use the function <head()>.

head(df)

To view the last six lines of df use the function <tail()>.

tail(df)

Maybe before these you’d like to know the number of rows in df. For this, use the <nrow()> function.

nrow(df)

To know the number of columns in df, use the <ncol()> function.

ncol(df)

To view a list of all of the defined data use the <ls()> function.

ls(df)

To view the type of variable you have e.g. integer, character, etc., use the <typeof()> function. I’m curious what type of variable page1 is in df.

typeof(df$page1)

Looking to view the summary statistics before you start manipulating your data? Use the <summary()> function.

summary(df)

Selecting Variables

Example: Maybe you’d like to choose the first 5 lines of your data frame and save them into a new data frame. I’m going to call this new data frame head.df. I would then simply use the <head()> function and specify the number of lines I want, like so:

head.df <- head(df, 5)

Maybe you’d like to select portions of this new data frame, head.df. Say these variable names are page1, page2, and page3 and you’d like to save them back into head.df. For this, use the <select()> function.

If they these columns are beside each other:

head.df <- select(head.df, page1:page3)

If they are not beside each other separate the variable names by a comma:

head.df <- select(head.df, page1, page2, page3)

Exploring Missingness

As there is inevitably some human error in many of our data frames, everyone should get comfortable with dealing with missing values. While this tutorial will be covering some basic ways to deal with missingness, I would recommend following this tutorial up with Allison Horst’s “Exploring missing values in naniar” tutorial.

For this portion of the tutorial, please download and call on the nanier package:

install.packages("naniar")
library(naniar)

To figure out how many missing data points you have in head.df, use the <miss_var_summary()> function. This gives you a table which summarizes all of your missing values in your data frame.

miss_var_summary(head.df)

If lists aren’t your thing and you’d prefer a quick and dirty visualization of the missingness of your data, use the <vis_miss()> function.

vis_miss(head.df)

Once you know whether or not you have missing values and where they are, you may want to exclude them from your analyses. Do to this, use the <na.rm> argument. In this case, we’re trying to find the mean of page1 in head.df without the missing values.

mean(head.df$page1, na.rm = TRUE)
